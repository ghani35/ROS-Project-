Description\+: ~\newline
 Control the robot depending on the {\bfseries{mode}} entred by the user. If the user choose {\bfseries{mode1}} he must set a goal point. The goal can be {\itshape canceled} by pressing \char`\"{}4\char`\"{} that takes you to {\bfseries{mode4}} and {\bfseries{mode3 and mode 4}} are for driving the robot by the {\itshape keyboard}.

This node is run by the launch file {\bfseries{solution.\+launch}} 
\end{DoxyAttention}

to have other {\itshape packages} and {\itshape nodes} running concurrently.
