Description: This node asks the user to enter an integer value. each ingere value points to a different mode where : 

    1 : Set a goalpoint 
    2 : Drive the robot by the keyboard *not assestive*
    3 : Drive the robot by the keyboard *assistive*
    4 : Cancel the goal

in case the user chose mode 1, he must set a goal point

Publishers to:
/user_topic

Attention
The goal point is in 2D
has the form (x,y), while x and y are floats
