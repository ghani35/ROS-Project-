Description: This node asks the user to enter an integer value. each integer value points to a different mode

Publishers to:
/user_topic

Note
These are the mode's reprentations of each integer value:
1 : Set a goalpiont
2 : Drive the robot by the keyboard Not assestive
3 : Drive the robot by hte keyboard Assistive
4 : Cancel the goal
Warning
in case the user chose mode 1, he must set a goal point
Attention
The goal point is in 2D
has the form (x,y), while x and y are floats
