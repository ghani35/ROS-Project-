Detailed Description

This is the controller node.

Author
Bakour Abdelghani bakou.nosp@m.rabd.nosp@m.elgha.nosp@m.ni19.nosp@m.99@gm.nosp@m.ail..nosp@m.com
Version
1.0
Date
25/12/2021

Publishers to:
/move_base/goal /move_base/cancel /cmd_vel

Subscribers to:
/user_topic /scan /cmd_vel /move_base/status

Actions :
/MoveBaseActionGoal /GoalID /GoalStatusArray

Description:
Control the robot depending on the mode entred by the user. There are four modes: mode1, mode2, mode3 and mode4 If the user choose mode1 he must set a goal point. The goal can be canceled by pressing "4" that takes you to mode4, in this mode the robot stops and do nothing, However mode2 and mode3 are used for driving the robot by the keyboard.

Note
This node must be run at the same time with the node [user_interface.cpp]
To make sure everything work correctly you can run the the launch file,
Solution.launch.
See also
user_interface.cpp
